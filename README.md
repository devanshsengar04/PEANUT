## 🌍 About The Project

**PEANUT** is a comprehensive environmental forecasting system designed to predict and highlight the "greenery deficit" across various sub-districts in India. By combining real-time satellite imagery with Artificial Intelligence, PEANUT aims to facilitate data-driven decision-making for future reforestation efforts, ultimately helping to meet the World Health Organization's per-capita green space targets by the year 2035.

### The Problem
As populations grow and urbanization expands, natural green cover is rapidly depleting. To maintain ecological balance and public health, it is essential to plan tree-planting initiatives *today* based on the population demands of *tomorrow*.

### The Solution
PEANUT provides a dual approach:
1. **Analytical Engine**: A Python-based spatial tracking tool that calculates current green cover and predicts future needs.
2. **Interactive Web Portal**: A Next.js dashboard where users can explore district maps, receive plant recommendations, and visualize environmental data.

---

## ⚙️ How It Works

PEANUT's architecture is divided into two primary wings:

### 1. Spatial Analysis & Predictive AI (`/Code`)
We utilize **Google Earth Engine (GEE)** to process massive datasets of satellite imagery and climate data for sub-districts in India.
* **Greenery Tracking**: Analyzes recent Sentinel-2 images using NDVI (Normalized Difference Vegetation Index) to map current vegetation.
* **Environmental Metrics**: Fetches and calculates mean annual rainfall, surface water, and soil organic carbon content.
* **AI Population Forecasting (Polynomial Regression)**: Uses historical population data (from WorldPop) and trains a Scikit-Learn Polynomial Regression model to forecast population metrics for 2035.
* **Deficit Calculation**: Compares the projected 2035 population target (9 sqm green space per capita minimum) against the current canopy target to highlight EXACTLY where the deficit lies.

### 2. The Web Dashboard (`/Website`)
Built on **Next.js**, the modern frontend provides an interactive experience.
* **Interactive Maps**: District visualization powered by GeoJSON.
* **Smart Recommendations**: Evaluates the environmental metrics generated by the AI to suggest climate-appropriate trees for specific districts.

## 🚀 Getting Started

### Running the Environmental Analysis (Python)
1. Ensure you have the `earthengine-api`, `pandas`, `sklearn`, and `numpy` installed.
2. Authenticate with Google Earth Engine using your project credentials.
3. Run `python Code/environmental_analysis.py`.
4. The script will batch process GADM shapefiles and output an excel file with 2035 predictions.

### Running the Web Platform
1. Navigate to the frontend: `cd Website`
2. Install dependencies: `pnpm install`
3. Start the dev server: `pnpm dev`
4. Jump into `http://localhost:3000` to interact with the map!
